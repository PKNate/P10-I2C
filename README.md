# P10-I2C
A turn and slip indicator on a PIC18F4550, using a gyroscope/accelerometer (MPU6050 GY-521) and a magnetometer (QMC5883 GY-273), shown on a 128x64 OLED screen (SSD1306).
Final project for Microsystems.
